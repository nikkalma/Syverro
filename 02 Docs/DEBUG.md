# История проблем и решений

## 2026-05-25 — Запуск лендинга syverro.com

- **Файлы/зоны:** DNS Nic.ru, проект Vercel, папка `syverro-landing`
- **Ошибки:** 
  1. Деплой из корня проекта → Vercel выложил `index.js` вместо HTML
  2. Домен был делегирован на `statuspage`, а не на `DNS-master`
- **Решение:** 
  1. `cd syverro-landing && vercel --prod`
  2. В Nic.ru: Домены → `syverro.com` → Изменить DNS → выбрать `DNS-master`
- **Статус:** ✅
- **Вывод:** DNS-пропагация занимает часы, а Nic.ru требует ручного переключения на `DNS-master` для работы кастомных записей.

## 2026-05-24 — Рейтинг 1–10 → 1–5
- **Ошибка:** `repeat` падал на числах 6–10 из Google Sheets
- **Решение:** Конвертация `ratingNum / 2` при импорте
- **Статус:** ✅

## 2026-05-24 — Ошибка repeat в карточке книги
- **Ошибка:** В `ViewMode.js` рейтинг 10 вызывал `RangeError`
- **Решение:** Защита `Math.min(5, Math.max(0, Number(...)))`
- **Статус:** ✅

## 2026-05-22 — Ачивка «Золотая звезда»
- **Ошибка:** Не разблокировалась (rating строка, ID не совпадал)
- **Решение:** `Number(b.rating) === 5`, синхронизирован ID `gold_star`
- **Статус:** ✅

## 2026-05-22 — Spread-оператор (массовая порча)
- **Ошибка:** `.{props}` вместо `{...props}` после автозамены
- **Решение:** Найти `.{` и `.[` → заменить на `...`
- **Статус:** ✅

## 2026-05-21 — Клавиатура (Android)
- **Решение:** `KeyboardAvoidingView` + `scrollToEnd` + запас отступа
- **Статус:** 🔄 (ожидает теста)

## 2026-05-20 — Экспорт JSON

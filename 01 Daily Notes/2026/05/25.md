## ⚡ Dev log

### Ночь

Была весьма продуктивной. 
- Диагностика: `syverro.com` показывал `index.js` вместо лендинга
- Причина: деплой из корня проекта, а не из `syverro-landing`
- Решение: `cd syverro-landing && vercel --prod`
- Анализ UX от ChatGPT: «продукт уже умный, но ещё не приятный жить каждый день»
- Решение: архитектурная реорганизация — выделение 4 когнитивных слоёв
- Создание `InsightsScreen`, очистка `HomeScreen`
- Массовая замена `lang.stats` → `lang.insights` по всему проекту
- Обновление локалей, добавление блока `insights`
- Настройка Obsidian: хранилище, шаблоны, дашборд
- Коммит всех изменений, отправка на GitHub

### Утро
- Аккаунты на ресурсах и в социальных сетях
- VPN на ноутбук для работы не только с телефона
- Концепт RIS и стратегия роста (Чат GPT)

### Вечер

- Ошибка лэндинга 404

# История проблем и решений

## 2026-05-20 — Оценка в режиме редактирования отображается криво

- **Файлы:** `screens/BookDetailsScreen.js`
- **Ошибка:**
  - Вместо 5 кнопок со звёздами — 8 или больше
  - Повторяется слово «Оценка»
  - Звёзды съезжают, дублируются
- **Решение:**
  - Заменить блок с `flexWrap: 'wrap', gap: 12` на `flexDirection: 'row', justifyContent: 'space-between'`
  - Каждой кнопке дать `flex: 1` и `marginHorizontal: 4`
  - Условие `editRating >= star` заменить на `editRating === star`
  - Отображать `{star} ⭐` вместо строки звёзд (компактнее)
- **Статус:** 🔄 (ожидает подтверждения)

---

## 2026-05-20 — Клавиатура перекрывает поля внизу экрана

- **Файлы:** `screens/BookDetailsScreen.js`
- **Ошибка:** При редактировании поля «Отзыв» или «Заметки» клавиатура закрывает кнопки «Сохранить / Отмена»
- **Решение:**
  - Добавить `keyboardDismissMode="on-drag"` в `ScrollView`
  - Увеличить `paddingBottom` в `contentContainerStyle` со 120 до 140
  - Добавить `onFocus` в `TextInput` с `scrollViewRef.current?.scrollToEnd()`
  - Обернуть всё в `KeyboardAvoidingView`
- **Статус:** 🔄 (ожидает подтверждения)

---

## 2026-05-20 — Слишком широкое боковое меню (50% экрана)

- **Файлы:** `App.js`
- **Ошибка:** `drawerStyle` имел `width: '50%'`, меню занимало полэкрана
- **Решение:** Изменить на `width: 260` и добавить `drawerType="front"`
- **Статус:** ✅ (исправлено)

---

## 2026-05-19 — Экспорт библиотеки не работал

- **Файлы:** `utils/exportImport.js`
- **Ошибка:** Код ожидал структуру `data.state.books`, но Zustand persist хранит книги в `data.books`
- **Решение:** Исправить чтение: `const books = persistedData.books || []`
- **Статус:** ✅ (исправлено)

---

## 2026-05-18 — Google Books API не находит русские книги

- **Файлы:** `store.js`
- **Ошибка:** API возвращает `undefined` или обложки на английские издания
- **Решение:** Удалить `fetchBookCover` и все вызовы. Перейти на ручной выбор обложек.
- **Статус:** 🔄 (код удалён, ручной выбор не добавлен — в приоритете 2)

---

## 2026-05-17 — Миграция оценок из 10-балльной шкалы

- **Файлы:** `store.js` (функция `migrateOldRatings`)
- **Ошибка:** Старые данные могли содержать оценки от 1 до 10, новые ожидают 1–5
- **Решение:** `const newRating = Math.round(book.rating / 2)` + защита от нуля
- **Статус:** ✅ (исправлено)

---

## 2026-05-16 — Импорт из Google Sheets (парсинг CSV)

- **Файлы:** `store.js` (функция `importBooksFromSheets`)
- **Ошибка:** CSV мог содержать кавычки, переносы строк внутри полей
- **Решение:** Написать ручной парсер с флагом `inQuotes` (без сторонних библиотек)
- **Статус:** ✅ (работает в DEV-режиме)
